# 4-ball-challenge
Procedural approach to play with 4 ball in processing
Here 4 balls are placed at the position relative to the height of the frame 

Ball ----- Position -------- Speed
B1         height/5.      1frames/sec
B2       2*height/5.      2frames/sec
B3       3*height/5.      3frames/sec
B4       4*height/5.      4frames/sec
